# SIAP-TEMAN

**Sistem Informasi Akses serta Terapi dan Monitoring Akses Pasien Dialisis yang Aman**

SIAP-TEMAN adalah aplikasi berbasis web yang dirancang untuk membantu tenaga kesehatan dan pasien dalam memantau kondisi akses dialisis secara aman, terstruktur, dan mudah digunakan.

## 🎯 Tujuan
- Meningkatkan keamanan akses dialisis (AV fistula, CVC, dll)
- Membantu monitoring kondisi pasien secara berkala
- Memberikan edukasi kepada pasien terkait perawatan akses
- Mendukung proses screening risiko (low, medium, high)

## 🔧 Fitur Utama
- Monitoring akses dialisis
- Edukasi pasien (visual / interaktif)
- Sistem penilaian risiko akses
- Panduan perawatan (care guideline)
- Interface sederhana dan mudah diakses

## 🌐 Platform
- Berbasis Web (GitHub Pages)
- Dapat digunakan di HP seperti aplikasi (Add to Home Screen)

## 📌 Target Pengguna
- Tenaga kesehatan (perawat, dokter)
- Pasien hemodialisis
- Keluarga pasien
